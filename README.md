# WebP Weight Reducer

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**WebP Weight Reducer** is a fast and secure web tool designed to convert JPG/PNG images into WebP format directly within your browser. Optimize your image file sizes for web performance in seconds without compromising privacy.

## 🚀 Features

- **Total Privacy**: Conversion happens entirely client-side. No data is ever uploaded to external servers.
- **Batch Optimization**: Upload dozens of images and convert them all with a single click.
- **Quality Control**: Granular slider to adjust the quality-to-weight ratio.
- **Auto-Resize**: Automatic resizing for images exceeding 1920px to maximize space savings.
- **ZIP Archive**: Export all results into a single, ready-to-use ZIP file.

## 🛠️ Technologies

- **HTML5/CSS3**: Modern, responsive "Dark Mode" interface.
- **JSZip**: Library for creating ZIP archives directly in the browser.
- **Canvas API**: For image rendering and re-encoding.

## 💻 Installation

No installation is required. Simply download the `Webp-converter.html` file and open it with any modern web browser.

```bash
# Clone the repository
git clone [https://github.com/your-username/webp-weight-reducer.git](https://github.com/your-username/webp-weight-reducer.git)

# Enter the directory
cd webp-weight-reducer

# Open Webp-converter.html in your browser
